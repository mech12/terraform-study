# 개요
* aws ami EC2 Instance 실행

# 준비
* ssh 키쌍 생성
```sh
ssh-keygen -t rsa -b 4096 -f demo
```

# 실행
* 약 1분 소요
```sh
terraform init
terraform apply
```
